## Information exchange specifications

Information exchange specifications document [R04] consists of the main document and underlying documents per IE.

These specifications cover the structure and the content of all the messages to be exchanged between the system components.

The document covers External domain IEs to be exchanged between the Economic operator and Customs administration via the TI.

There are 84 IE specified.
