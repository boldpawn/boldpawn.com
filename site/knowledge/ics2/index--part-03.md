## Routing guide — which file answers which question

| If the question is about… | Look in… |
| --- | --- |
| Scope, structure, release notes, document list, abbreviations | `ICS2-HTI-Main-(2022-05-23)-v2.02.md` |
| Definition of a term / acronym used across ICS2 | `ICS2-HTI-Definitions-(2020-03-16)-v1.11.md` |
| Business process / who does what / end-to-end flow (business view) | `3. ICS2-HTI-BPML3.5-…/` (BPM L3.5 — see catalog) |
| System/IT process flow, tasks, gateways, which IE is sent when | `4. ICS2-HTI-BPML4-…/` (BPM L4 — see catalog) |
| The data content of a message (elements, cardinality, format, codes, XPath) | `5. …/publish_XLS_…/IE3xxx.md` (IE catalog) |
| Overview of the IE specifications, notation, list of IEs, amendment principles | `5. …/4. ICS2-HTI-IE-(2022-05-04)-v2.02.md` |
| A data rule (R-code, e.g. R3024) or condition (C-code, e.g. C3001) | `ICS2-HTI-R&C-(2022-05-23)-v2.01.md` |
| A code list (CL-code, e.g. CL742) and its values | `ICS2-HTI-CL-(2022-05-23)-v2.01.md` |

> The IE message files cross-reference Rules (`Rules` column → R&C doc), Conditions (`Conditions` column → R&C doc), and Code Lists (`ICS2 CL` columns → CL doc). A full answer about a message element often needs the IE file **plus** the R&C and/or CL doc.
