## BPM L4 Process Description

BPM L4 Process Description document [R03] provides a mapping of the business process and data to the IT system components.

There are 3 components of the ICS2:

- Trader interface

- Common Repository

- National Entry Systems

BPM L4 depicts the process at TI component, interface with CR as well as information exchanges between the 2 components. The process is depicted through the sequence of the system tasks and exchanges of messages. There is 1 Master process and 8 underlying sub-processes.

BPM L4 also provides 1 supporting process which will offer additional functionality to the economic operators, such as consultation for specific ENS filings by the economic operators.
