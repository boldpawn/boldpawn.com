## Purpose

This document provides the ICS2 Information exchange specifications. These specifications are based on the Annex B of the UCC DA and IA [A0**2**] as well as EU Customs Data Model (EUCDM) [A02] and are depicted in the ICS2 BPML4 process description [A03].

All IE specifications will be implemented in a harmonised way across all different ICS2 IT components and different implementations of TI.

This document serves as a cover and explanatory note for IE specifications which are provided in annexes in individual document per IE.
